ID Wilayah API

API Documentation & Playground

A simple, modern API for Indonesian wilayah (administrative region) data—provinces, cities, districts, and villages.
Built for developers who need up-to-date, structured wilayah info for forms, dashboards, or address validation.

Available Endpoints

  • GET /states — List all provinces (states)
  • GET /states/:stateCode/cities — List all cities/regencies in a province
  • GET /cities/:cityCode/districts — List all districts in a city/regency
  • GET /districts/:districtCode/villages — List all villages in a district

How to Use

Simply send a GET request to the desired endpoint. Example using curl:

curl https://id-wilayah-api.gilangpratama.id/states

Or from JavaScript:

fetch('https://id-wilayah-api.gilangpratama.id/states')
  .then(r => r.json())
  .then(console.log);
